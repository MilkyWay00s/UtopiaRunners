# UtopiaRunners
밀키웨이 팀의 러닝액션게임 레포지토리입니다.

## commit convention

|태그|설명|
|---|---|
|feat|새로운 코드 추가|
|fix|문제점 수정|
|refact|코드 리팩토링|
|balance|밸런스 및 수치 조정|
|comment|주석 추가(코드 변경X) 혹은 오타 수정|
|asset|이미지, 사운드 등 게임 리소스 추가/수정|
|docs|README와 같은 문서 수정|
|merge|merge|
|rename|파일, 폴더명 수정 혹은 이동|
|remove|불필요한 코드나 파일, 폴더 삭제|
```
ex) feat(파일명)
    시작 이벤트 구현
```
